🎵 미디어 페이지

W3Schools 기반 — audio, video 태그를 활용한 미디어 페이지

① audio 태그 — 기본 사용 (controls)

controls 속성을 사용하여 재생 컨트롤러를 표시합니다.

※ W3Schools 제공 예제 오디오 파일 (horse.ogg / horse.mp3)을 사용합니다.

② audio 태그 — autoplay + muted 속성

autoplay는 페이지 로드 시 자동 재생, muted는 음소거 상태로 시작합니다. (브라우저 정책상 muted 없이 autoplay는 차단될 수 있습니다.)

③ audio 태그 — loop 속성

loop 속성을 사용하면 오디오가 끝난 후 처음부터 반복 재생됩니다.

④ video 태그 — 기본 사용 (controls)

controls 속성으로 재생, 일시정지, 볼륨 등 컨트롤러를 표시합니다.

※ W3Schools 제공 예제 동영상 파일 (mov_bbb.mp4)을 사용합니다.

⑤ video 태그 — width, height, autoplay, muted

width, height로 크기를 지정하고, autoplay와 muted를 함께 사용합니다.

⑥ video 태그 — poster 속성 (썸네일 이미지)

poster 속성으로 동영상 재생 전 표시될 썸네일 이미지를 지정합니다.

⑦ YouTube 동영상 삽입 — iframe 태그

<iframe> 태그를 사용하여 YouTube 영상을 페이지에 삽입할 수 있습니다.

※ YouTube embed URL 형식: https://www.youtube.com/embed/[영상ID]

⑧ audio / video 주요 속성 정리

속성 설명 audio video
controls 재생 컨트롤러 표시 ✅ ✅
autoplay 자동 재생 (muted 권장) ✅ ✅
muted 음소거 상태로 시작 ✅ ✅
loop 반복 재생 ✅ ✅
poster 재생 전 썸네일 이미지 ❌ ✅
width / height 크기 지정 ❌ ✅
