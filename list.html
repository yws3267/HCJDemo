📋 List 태그 예제

W3Schools 기반 — ul, ol, dl 태그를 활용한 목록 페이지

① Unordered List (ul) — 순서 없는 목록

기본 스타일(disc)을 사용한 순서 없는 목록입니다.

  • HTML (HyperText Markup Language)
  • CSS (Cascading Style Sheets)
  • JavaScript
  • Python
  • Java

② Unordered List — circle 스타일

  • React
  • Vue.js
  • Angular
  • Svelte

③ Unordered List — square 스타일

  • MongoDB
  • MySQL
  • PostgreSQL
  • SQLite

④ Ordered List (ol) — 순서 있는 목록 (숫자)

  1. VS Code 설치
  2. HTML 파일 생성 (index.html)
  3. 기본 구조 작성
  4. 브라우저에서 열기
  5. 결과 확인

⑤ Ordered List — 알파벳 스타일 (type="A")

  1. 기획
  2. 디자인
  3. 개발
  4. 테스트
  5. 배포

⑥ Ordered List — 로마자 스타일 (type="I")

  1. 요구사항 분석
  2. 시스템 설계
  3. 구현
  4. 통합 및 테스트
  5. 유지보수

⑦ Nested List — 중첩 목록

  • 프론트엔드
    • HTML
    • CSS
    • JavaScript
  • 백엔드
    • Node.js
    • Django
    • Spring
  • 데이터베이스
    • MySQL
    • MongoDB

⑧ Description List (dl) — 설명 목록

HTML
웹 페이지의 구조와 내용을 정의하는 마크업 언어입니다.
CSS
웹 페이지의 스타일(색상, 레이아웃, 폰트 등)을 지정하는 언어입니다.
JavaScript
웹 페이지에 동적 기능과 상호작용을 추가하는 프로그래밍 언어입니다.
GitHub
코드를 버전 관리하고 협업하는 데 사용되는 플랫폼입니다.
Vercel
프론트엔드 프로젝트를 간편하게 배포할 수 있는 클라우드 플랫폼입니다.
